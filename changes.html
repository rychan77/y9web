Changes to my design:

After proceeding and completing my developing ideas page, there were some designs that seemed quite hard to do. However, I underestimated myself and I completed that part with time to spare. Furthermore, I decided to add a few more features that would help the aesthetics and make the page more modern. The initial design had many features within different pages. I’ll show the initial design before and what I coded in turn of that below.

Homepage:

The homepage had a navigation bar across the top with icons on it such as logo, which would bring you back to the homepage on click. The data icon when hovered over dropped all the countries. The need help? part when hovered dropped down 2 different selections: assessing the situation and comforting the victim. Lastly, in the nav bar, there was a get help now button that would lead to a page that would have a suicide crisis hotline number on it.

The nav bar I coded looks almost identical to the one designed however, the buttons and icons have a sign on them. The logo icon when clicked on opens the side nav bar which you can access any part of the website. This was done because it faster to go straight to the page instead of back to the homepage and then to the wanted page. Next, the data drop down box is still a drop down but has 2 different options: Globe data and graph data. I did this because I wanted different ways of accessing the data for accessibility reasons. Alongside the data dropdown was the need help drop down. There were no changes made to the initial design. Lastly, the get help now button has the same function however, on hover, there is a slight shadow to make the website more extravagant.

The initial content of the homepage was supposed to be bland and rather boring. The big image was supposed to be a lonely person which matches the theme of the page of suicide and depression. Below the big image was supposed to be a section about the page.

Honestly, when looking back at the design plan, I found the homepage content awful. I wanted to make the page nicer and more features. So in turn, I put in parallax scrolling. PArallax scrolling is where the mage changes when text breaks the images. It looks cool and mesmerizing in my opinion. I changed the main photo to a hand helping another hand up. While the theme is supposed to be suicide and depression all that dark stuff, I want to make the website less depressing and more supportive. I removed the about section from the homepage and wrote a bit about what suicide is and a brief sentence on what this website’s purpose is. Overall, the homepage has lots of design changes to make it more aesthetically pleasing.

Data page:

The data page’s content was rather simple and the graphs would change based on the chosen country in the scroll down bar. The country name would appear in the top left corner in big text and then there would be 6 buttons, 3 a side and on click, a graph would appear beside it corresponding to the content from the data.json data.

However, due to changes made in the nav bar, the countries would now be selected in on the “graph data” page in the top left corner in a scroll drop bar. After a country is chosen the name of the country is placed below the scroll bar. Next, the 6 categories would be lined up in one instead of split into 2 groups of 3. There will be no buttons and after the country is selected in the scroll bar, instead, the graphs would appear automatically. I removed the buttons because I feel like it would be too much of a hassle to press all 6 buttons each time a different country was selected. However, I added an svg map of the world and when you click, the name of the country appears to help identify the proper country. In addition, I added the globe data part because I wanted different ways to access the data and it is visually nice.

Get Help Now page:

The original get help now page had the title across the top and a little caption below in smaller font. Below, there was a table with 3 different of methods of reaching a suicide crisis hotline, by phone, by email, and by text. These 3 choices were in a table all in a row. The background would be plain sky blue

The new design coded is practically the same however, the background of the website would be a nice blue background with designs within instead of just a plain sky blue background. In addition, instead of using a table, I used flex boxes instead because I wanted there to be space between the 3 choices not all tight and crammed together. Furthermore, the text and images blended with the chosen background so flex boxes with a white grey background would be better to read and contrast the text and images.

List pages:

Both of the list pages were supposed to be identical with just text of a list of phrases and questions, as well as an image beside that.

Instead, I created 2 different list pages because I wanted some more script and more funky features instead of just plain text and a bland image. On my assessing the page, I put a picture slideshow of suicidal people being comforted by others. With this, I put some questions to assess the situation on top of the slideshow images. The images change every 5 seconds. Next, the comforting the victim page shows 6 different images and when you hover over them, the image has a blue screen go over the image with different phrases to help the victim.

In conclusion, to justify these changes, I underestimated my coding skills and during this project, I exceeded my expectations and learned a lot about html, css and a little about script however, I really enjoyed coding with css and styling my page so it looks somewhat aesthetically pleasing. From this, I created extra sub headings to make the website easy to navigate. Overall, for my first actual website, this website is actually not bad.
